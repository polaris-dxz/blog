# 分享几个常用的插件工具

## 一、gem

### 1、synx 

1. 介绍：

   synx：同步 xcode 工程中的 group 至当前的 finder 中

2. 安装：

   ```shell
   $ gem install synx
   ```


3. 使用：

   ```shell
   $ synx your/product/path/project.xcodeproj #项目中有 pod 时一样用xcodeproj
   ```

4. 注意事项：

   导入 .a 和 .framework文件时，或者 支付宝 SDK 时可能会出现一些问题
